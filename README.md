# Hello World

Open [index.html](index.html) in your browser, or serve this folder and visit http://localhost:8000

This version includes visual enhancements:

- animated soft background blobs
- a friendly waving emoji and time-aware greeting
- a "Surprise me" button that cycles messages and emits particles
- a Dark/Light theme toggle (top-right)

To serve with Python (if installed):

```powershell
python -m http.server 8000
```
